# E-Commerce Sales & Inventory Analytics

## Project Overview
This project analyzes e-commerce sales and inventory data to derive business insights such as total revenue, top-performing products, revenue by category, and city-wise sales trends.

## Tools Used
- Python (Pandas, NumPy)
- Jupyter Notebook
- Power BI
- CSV datasets

## Key Analysis
- Data cleaning and merging of orders, products, and customers
- Revenue calculation and KPI analysis
- Top products and categories by revenue
- Geographic (city-wise) revenue analysis

## Dashboard
Power BI dashboard includes:
- Total Revenue KPI
- Top Products by Revenue
- Revenue by Category
- Top Cities by Revenue

## Outcome
This project demonstrates end-to-end data analytics workflow from raw data to interactive dashboard insights.
